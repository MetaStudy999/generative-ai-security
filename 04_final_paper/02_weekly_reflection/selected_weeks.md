# 선택 주차 및 전체 반영 범위

## 1. 기말 논문 반영 대상

기말 논문은 W01-W15 final 보고서 전체를 반영 대상으로 둔다. 다만 논문의 중심 주제는 LLM/RAG·프라이버시·재현성 기반 AI 보안 평가 프레임워크이므로, W07, W08, W11, W14, W15를 핵심 축으로 삼고 나머지 주차는 공통 이론 배경, 관련연구, 보안 평가 지표, 한계 분석에 연결한다.

| 주차 | 주제 | 기말 논문 연결 | 반영 구분 |
|---|---|---|---|
| W01 | 딥러닝 패러다임 & ML 보안 분류학 | ML 생명주기 보안 분류의 기초 | 공통 배경 |
| W02 | 대규모 최적화 & 데이터 오염 위협 | poisoning/backdoor 평가 지표 | 공통 배경 |
| W03 | 컴퓨터비전 & 대적 공격 | adversarial robustness와 무결성 위협 | 공통 배경 |
| W04 | Transformer/NLP 보안 | attention, NLP robustness, prompt privacy | 공통 배경 |
| W05 | SSL & backdoor | 표현학습과 backdoor stealthiness | 공통 배경 |
| W06 | Diffusion/GAN & deepfake | 생성모형 안전성과 탐지 신뢰성 | 공통 배경 |
| W07 | LLM 보안·프라이버시 | LLM 평가, 보안·프라이버시 위협 | 핵심 축 |
| W08 | RAG·프롬프트 인젝션 | RAG 공격면과 source verification | 핵심 축 |
| W09 | DRL 사이버보안 | 보상 설계와 safety violation | 보조 평가축 |
| W10 | 연합학습 보안 | FL privacy, poisoning, robust aggregation | 보조 평가축 |
| W11 | 차등프라이버시 & MI | privacy leakage와 utility trade-off | 핵심 축 |
| W12 | NN 검증 & XAI | certified/empirical robustness, 설명 신뢰성 | 보조 평가축 |
| W13 | 모델 도난 & 워터마킹 | extraction fidelity와 ownership verification | 보조 평가축 |
| W14 | MLOps/공급망 보안 | 재현 가능한 파이프라인과 artifact 관리 | 핵심 축 |
| W15 | 연구평가·재현성·XAI·논문 구성 | 연구윤리, 참고문헌 검증, AI 고지 | 핵심 축 |
