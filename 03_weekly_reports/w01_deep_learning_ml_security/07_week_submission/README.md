# W01 주차 제출 폴더

이 폴더는 딥러닝 패러다임 & ML 보안 분류학 주차의 제출본 점검표를 관리한다.

## 제출 기준

- 최종 보고서는 `06_report/final/integrated_report_final.md`를 기준으로 한다.
- 제출용 주차 보고서는 `07_week_submission/w01_submission_report.md`와 `07_week_submission/w01_submission_report.html`을 기준으로 한다.
- 참고문헌 검증은 `01_papers/doi_check.md`를 기준으로 한다.
- 실험은 synthetic toy evaluation으로 실행 완료했으며, 결과는 `04_experiment/outputs/`를 기준으로 한다.
- 발표자료는 `09_presentation/`의 슬라이드, 발표자 노트, Q&A, 1페이지 요약을 기준으로 한다.
