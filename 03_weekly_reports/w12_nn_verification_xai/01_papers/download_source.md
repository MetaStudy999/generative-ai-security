## public GitHub 보관 정책

- `git ls-files` 기준 PDF 5개가 이미 추적 중이다.
- `.gitignore`에는 `03_weekly_reports/**/01_papers/pdf/*.pdf`가 포함되어 있다.
- 이미 커밋된 PDF는 `.gitignore`만으로 제거되지 않는다.
- 공개 저장소에는 원칙적으로 PDF 원문 대신 DOI/URL, 서지정보, 요약, 확인 로그만 남긴다.
- PDF 삭제 또는 history 정리는 사용자 승인 후 별도 작업으로 수행한다.
