1-4. 요약과 핵심 내용

모델 추출 위험은 query budget과 extraction fidelity로, 워터마크 기반 소유권 검증은 watermark detection과 false positive rate를 함께 보아야 한다. 모델 도난과 모델 추출은 query-response 정보를 통해 모델 행동이나 결정경계를 모방하려는 공격군으로 분류된다[1]. Watermarking과 fingerprinting은 소유권 검증 신호를 남기는 접근이지만 detection rate와 false positive를 함께 봐야 한다[2][3][4].

6-7. 비교와 Research Track

P01은 model stealing/extraction taxonomy, P03은 DNN watermarking 요구조건, P04는 extraction 이후 ownership check를 담당한다. P02와 P05는 로컬 PDF가 지정 논문과 불일치하므로 보조 문헌으로만 사용한다.

False Positive 해석

Watermark detection은 일부 조건에서 1.000000이지만 false positive proxy도 0.600000으로 높다. 이는 trigger-set 기반 소유권 검증이 detection rate만으로는 충분하지 않으며, clean control model, unrelated model, random trigger set, 복수 seed, 통계적 유의성 검정이 필요함을 의미한다.

12-14. KCI/SCI/발표 연결

KCI 추천 제목은 “모델 추출 이후 소유권 검증을 위한 다중지표 평가 프레임워크 연구”이며, SCI 제목 후보는 “A Multi-Metric Evaluation Framework for Model Extraction Risk and Watermark-Based Ownership Verification”이다. 발표자료는 09_presentation/에 정리했다.
