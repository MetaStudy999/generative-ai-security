# AI 활용 고지서

| 항목 | 작성 내용 |
|---|---|
| 사용한 AI 도구명 | Codex, ChatGPT 등 실제 사용 도구 |
| 사용 일자 | 2026-06-22 또는 확인 필요 |
| 사용 목적 | 논문 요약 보강, 개념 설명, 보고서 구조화, DOI/URL 검증표 정리, 실습 소스 작성, 제출용 보고서 정리 |
| 주요 프롬프트 1 | W01 보고서 최종 보완 지시 |
| 주요 프롬프트 2 | P04 논문 검증 및 참고문헌 정리 요청 |
| 주요 프롬프트 3 | KCI/SCI 논문 형식 전환 섹션 작성 요청 |
| AI 산출물 반영 위치 | W01 하위 Markdown, 실험 코드, 보고서 파일 |
| 본인 수정 내용 | 논문별 주장 분리, DOI/URL 검증, 실험 범위 제한, 인용 대응 확인 |
| 사실관계 검증 방법 | DOI/arXiv/출판사 페이지, 로컬 PDF 메타데이터, 실행 로그, 수업자료 대조 |
| 참고문헌 검증 방법 | 제목, 저자, 연도, 학술지, DOI/URL 대조 |
| 실험결과 처리 | outputs 파일과 보고서 수치 대조 |
| 아직 검토 필요한 항목 | P04 최종 출판정보, 국내 참고문헌, PDF 저작권 상태 |
| 최종 책임 확인 | 최종 제출자는 원고의 내용, 인용, 실험결과, 연구윤리 책임을 확인해야 한다. |

## 사용상 한계

- AI가 생성한 문장은 제출 전 초안으로만 취급한다.
- 실행하지 않은 실험 결과, 확인하지 못한 DOI/URL, 확인하지 못한 출판 정보는 확정 표현으로 쓰지 않는다.
- P04는 arXiv 논문 기준으로 정리되어 있으며 강의계획서 지정 IEEE COMST 논문과 동일 여부가 `확인 필요`다.
