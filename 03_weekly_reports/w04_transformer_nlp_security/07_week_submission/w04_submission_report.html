초록과 키워드

본 보고서는 Transformer 변형과 NLP 보안 위협을 함께 다룬다. AI 원리 측면에서는 self-attention, multi-head attention, sparse attention, low-rank approximation, kernelized attention 등 Efficient Transformer 계열의 계산 복잡도 완화 전략을 정리한다. 보안 측면에서는 단어 치환 기반 NLP 대적공격, 프롬프트 프라이버시, ICL 환경의 민감정보 노출, privacy-preserving prompt engineering을 분석한다.

키워드: Transformer, X-former, sparse attention, NLP adversarial attack, prompt privacy, ICL

기말논문 연결

추천 주제는 “프롬프트 기반 AI 시스템의 민감정보 보호 평가체계”이다. 기여 후보는 prompt privacy threat model, masking/policy control checklist, clean score와 privacy leakage의 동시 평가표, 실행 로그 기반 재현성 기준이다.
