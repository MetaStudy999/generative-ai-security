# 제출 전 최종 점검표

| 확인 | 점검 항목 |
|---|---|
| ☑ | 한국산학기술학회논문지 양식을 다운로드했다. |
| ☑ | 다운로드 URL과 확인일을 기록했다. |
| ☑ | 투고규정과 참고문헌 형식 확인 URL을 기록했다. |
| ☑ | 국문 제목과 영문 제목을 작성했다. |
| ☑ | 국문초록과 영문초록 초안을 작성했다. |
| ☑ | 국문·영문 키워드를 작성했다. |
| ☑ | 서론, 관련연구, 연구문제, 연구방법, 분석, 보안적 함의, 결론 구조를 포함했다. |
| ☑ | 표 1개 이상, 그림 1개 이상을 포함했다. |
| ☑ | 국내 논문 3편 이상, 해외 논문 5편 이상을 검증표에 기록했다. |
| ☑ | 목표 기준 이상인 국내 6편, 해외 8편을 검토했다. |
| ☑ | 주차별 보고서 2개 이상을 논문에 반영했다. |
| ☑ | AI 활용 고지서를 작성했다. |
| ☑ | 참고문헌 검증표를 작성했다. |
| ☑ | synthetic 데이터셋이 실제 개인정보를 포함하지 않는지 확인했다. |
| ☑ | 실험 결과가 실제 실행 결과인지 확인했다. |
| ☑ | 허위 DOI 또는 존재하지 않는 참고문헌이 없는지 확인했다. |
| ☑ | DOCX 초안을 생성했다. |
| ☑ | PDF 초안을 생성했다. |
| □ | DOCX 또는 HWP 원본을 공식 JKAIS 양식에 맞게 최종 편집했다. |
| □ | 최종 PDF 변환 후 서식 깨짐을 육안 확인했다. |
| □ | 최종 파일명을 규칙에 맞게 작성했다. |

## 남은 수동 점검

- HWP 공식 양식에는 제출자가 직접 저자명, 소속, 학번, 교신저자 정보를 입력해야 한다.
- 자동 생성된 DOCX/PDF 초안은 공식 JKAIS HWP 조판과 다를 수 있으므로 최종 제출 전 수동 검수한다.
