초록

본 보고서는 RAG, GraphRAG, prompting framework의 기본 구조를 정리하고 prompt injection, indirect injection, RAG 문서 오염, tool misuse를 분석한다. Synthetic RAG document와 rule-based toy evaluator로 source filter와 human approval gate의 효과를 측정했다. Poisoned document 조건의 ASR은 0.575000이었고 Source filter 적용 후 0.050000, Human approval 적용 후 0.025000으로 낮아졌다.

키워드: RAG, GraphRAG, prompt injection, indirect injection, source verification, tool misuse, human approval

AI 원리 70%와 보안 이슈 30%

RAG는 외부 근거를 검색해 LLM context에 넣고 답변을 생성한다. GraphRAG는 node, edge, path, subgraph 같은 관계 구조를 활용한다. 보안상 핵심 문제는 검색 context와 tool instruction의 trust boundary가 흐려질 때 indirect injection과 tool misuse가 발생한다는 점이다.

기말논문 연결 및 AI 활용 고지

추천 주제는 RAG 기반 생성형 AI 시스템에서 간접 프롬프트 인젝션 대응을 위한 출처 검증·승인 게이트 평가 프레임워크다. Codex를 사용해 공통 지침 확인, PDF 기반 서지 보정, synthetic 실험 코드 작성과 실행, 제출용 보고서 및 발표자료 작성을 수행했으며 정량값은 outputs 3종과 일치하는 값만 사용했다.
