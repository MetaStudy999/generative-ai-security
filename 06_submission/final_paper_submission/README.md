# 최종 제출 패키지

## 1. 포함 예정 파일

| 파일 | 상태 | 비고 |
|---|---|---|
| 최종 논문 DOCX | 확인 필요 | 선택 학회지 양식 적용 필요 |
| 최종 논문 PDF | 확인 필요 | DOCX 변환 후 서식 확인 |
| AI 활용 고지서 | 확인 필요 | 04_final_paper/06_appendices/ai_disclosure.md 기반 |
| 참고문헌 검증표 | 확인 필요 | reference_verification.md 기반 |
| 학회지 양식 출처표 | 확인 필요 | journal_format_source.md 기반 |
| 주차별 보고서 반영표 | 확인 필요 | 04_final_paper/02_weekly_reflection/ 기반 |

## 2. 제출 전 최종 점검

- [ ] 선택한 국내 학회지 양식을 사용했다.
- [ ] 국문 제목과 영문 제목을 작성했다.
- [ ] 국문초록과 영문초록을 작성했다.
- [ ] 키워드를 작성했다.
- [ ] 표 1개 이상을 포함했다.
- [ ] 그림 1개 이상을 포함했다.
- [ ] 국내 논문 3편 이상을 검증했다.
- [ ] 해외 논문 5편 이상을 검증했다.
- [ ] AI 활용 고지서를 작성했다.
- [ ] 참고문헌 검증표를 작성했다.
- [ ] 학회지 양식 출처표를 작성했다.
- [ ] PDF 변환 후 서식 깨짐을 확인했다.
