# W03 BibTeX 폴더

이 폴더는 W03 논문 패킷의 참고문헌 BibTeX 초안을 보관하기 위한 자리다.

현재 W03 제출본은 `01_papers/paper_list.md`와 `01_papers/doi_check.md`를 기준으로 참고문헌 검증 상태를 관리한다. DOI/URL을 공식 출처에서 확인하기 전에는 BibTeX 항목을 확정하지 않는다.

## 원칙

- DOI, URL, 학술지 권호, 페이지는 임의로 채우지 않는다.
- BibTeX를 추가할 경우 `doi_check.md`의 검증 상태도 함께 갱신한다.
- 최종 제출 전 학교/학회 양식에 맞춰 저자명, 연도, 제목 표기를 대조한다.
