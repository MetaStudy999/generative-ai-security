# W02 발표자료

## 구성

| 파일 | 용도 |
|---|---|
| `presentation_report.md` | 발표용 보고서 원본 |
| `presentation_report.html` | 브라우저 인쇄/PDF 저장용 발표 보고서 |
| `presentation_slides.md` | 공통 파일명 기준 발표 슬라이드 원본 |
| `presentation_slides.html` | 공통 파일명 기준 브라우저 발표용 HTML |
| `w02_presentation_slides.md` | W02 별칭 슬라이드 원본 |
| `w02_presentation.html` | W02 별칭 브라우저 발표용 HTML |
| `speaker_notes.md` | 슬라이드별 발표자 대본 |
| `qna.md` | 예상 질문과 답변 |
| `one_page_handout.md` | 청중 배포용 1페이지 요약 |

## 발표 시간 기준

- 권장 발표 시간: 8-10분
- 슬라이드 수: 14장, 표지 포함
- 핵심 메시지: 데이터 오염은 학습 목적함수와 gradient를 바꾸며, backdoor 평가는 clean accuracy와 ASR을 분리해야 한다.

## 사용 방법

1. `presentation_slides.html` 또는 `w02_presentation.html`을 브라우저로 연다.
2. 키보드 방향키, PageUp/PageDown, Home/End로 슬라이드를 이동한다.
3. 인쇄 또는 PDF 저장이 필요하면 브라우저 인쇄 기능을 사용한다.
