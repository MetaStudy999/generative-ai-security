W03 AI Security

컴퓨터비전 표현학습 & 비전 대적공격

Clean accuracy, robust accuracy, ASR, 재현성 근거를 분리해 평가한다.

좋은 표현학습이 곧 안전한 모델을 의미하지 않는다.
Problem

왜 W03가 중요한가

  • 이미지 모델은 분류, 검색, 인식, 멀티모달 AI의 핵심 구성요소다.
  • 정상 테스트셋에서 잘 맞아도 작은 입력 교란에 취약할 수 있다.
  • 핵심 질문: “모델이 이미지를 잘 본다”는 말은 보안적으로 충분한가?
성능 중심 비전 이해를 보안성 중심 평가로 확장한다.
Roadmap

AI Principle

CNN: 계층적 시각 표현

  • 지역 receptive field와 convolution으로 이미지 특징을 추출한다.
  • Pooling은 공간 정보를 압축하고 feature map을 안정화한다.
  • 얕은 계층은 edge/texture, 깊은 계층은 shape/object 표현을 학습한다.
입력 픽셀의 작은 변화도 feature map과 decision boundary를 바꿀 수 있다.

Experiment

Toy 실험 설계

  • 데이터: synthetic 8x8 vertical/horizontal bar image
  • 모델: nearest-centroid classifier
  • 공격: 반대 클래스 중심점 방향 L-infinity perturbation
  • 방어 점검: 2-level feature squeezing
  • 출력: CSV, JSON, Markdown run log, PGM 예시 이미지

Closing

결론

  • 비전 모델의 정상 성능은 보안성을 보장하지 않는다.
  • 공격 조건 성능과 ASR을 별도로 기록해야 한다.
  • 정량값은 실행 로그, config, seed, CSV/JSON이 있을 때만 주장한다.
  • W03는 재현성 중심 평가 프레임워크를 보강하는 사례다.
