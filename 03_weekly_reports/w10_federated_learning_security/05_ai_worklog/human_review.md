# 사람 검토 기록

## 1. 검토가 필요한 항목

| 항목 | 상태 | 메모 |
|---|---|---|
| P01 DOI | 확인 | 로컬 PDF 첫 페이지 기준 `10.1145/3678182` |
| P02 DOI | 확인 | 로컬 PDF 첫 페이지 기준 `10.1016/j.future.2020.10.007` |
| P03 출판 DOI | 확인 필요 | 로컬 PDF는 arXiv preprint |
| P04 DOI | 확인 | 로컬 PDF 참고문헌 형식 기준 `10.1145/3724113` |
| P05 출판 DOI | 확인 필요 | 로컬 PDF는 arXiv preprint |
| 실험 수치 | 확인 | `run_log.md`, `metrics_summary.csv`, `results.json` 기준 |

## 2. 사람 수정 권장

- 최종 제출 전 DOI landing page와 출판사 페이지를 다시 열어 참고문헌 형식을 확정한다.
- Privacy Leakage Proxy가 실제 privacy attack 성공률이 아니라는 설명을 발표 중 명확히 말한다.
- 기말논문에 W10을 반영할 경우 toy 실험의 한계를 먼저 적고, 방법론의 예시로만 사용한다.
