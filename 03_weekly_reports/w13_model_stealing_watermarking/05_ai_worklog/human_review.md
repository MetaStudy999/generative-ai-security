# 사람 검토 기록

## 검토 완료 항목

- W13 공통 지침 확인: 실행하지 않은 수치 금지, DOI 임의 생성 금지, toy 실험 제한
- 실험 수치 출처 확인: `outputs/metrics_summary.csv`, `results.json`, `run_log.md`
- 안전 범위 확인: 실제 API, 실제 개인정보, 무단 대량 질의, 실제 모델 탈취 절차 제외
- 대체 PDF 표시: P02와 P05는 프롬프트 지정 문헌과 로컬 PDF가 다름

## 검토 필요 항목

- DOI/URL 공식 출처 대조
- P02/P05를 프롬프트 지정 원문으로 교체할지, 대체 문헌으로 유지할지 결정
- 원문 세부 실험 설정과 수치 검증
- 기말 논문에 실제 인용할 문헌 선별
- AI가 작성한 표현의 과장 여부 최종 점검

## 현재 판단

W13 산출물은 제출 가능한 구조로 완성했으며, 실험 결과는 로컬 synthetic toy 평가로만 해석해야 한다. false positive proxy가 0.600000으로 높으므로 워터마크 검출 결과를 강한 소유권 증거로 과장하지 않는다.
