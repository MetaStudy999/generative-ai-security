# Local Fonts

| 파일 | 패밀리 | 출처 | 라이선스 | 사용 위치 |
|---|---|---|---|---|
| `NotoSansCJKkr-Regular.otf` | Noto Sans CJK KR | https://github.com/notofonts/noto-cjk | SIL Open Font License 1.1 | `scripts/generate_formula_visual_supplements.py`의 matplotlib 차트 렌더링 |

이 폰트는 system-wide 설치가 아니라 저장소 로컬 자산이다. 시스템 `apt` 설치는 현재 환경에서 `sudo` 비밀번호가 필요해 수행하지 못했다.
