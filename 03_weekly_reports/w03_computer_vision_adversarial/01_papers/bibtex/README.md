# W03 BibTeX 폴더

이 폴더는 W03 논문 패킷의 참고문헌 BibTeX 초안을 보관하기 위한 자리다.

현재 W03 제출용 최종 초안은 `01_papers/paper_list.md`와 `01_papers/doi_check.md`를 기준으로 참고문헌 검증 상태를 관리한다. P01-P05 DOI/URL은 2026-06-22 Crossref/DOI 기준으로 확인했지만, BibTeX 최종 양식은 제출 직전 학회/학교 양식에 맞춰 사람이 대조한다.

## 원칙

- DOI, URL, 학술지 권호, 페이지는 `doi_check.md`와 맞춘다.
- BibTeX를 추가할 경우 `doi_check.md`의 검증 상태도 함께 갱신한다.
- 최종 제출 전 학교/학회 양식에 맞춰 저자명, 연도, 제목 표기를 대조한다.
