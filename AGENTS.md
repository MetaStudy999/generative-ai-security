# AGENTS.md

## Repository Structure Rule

이 저장소는 번호형 루트 구조를 표준으로 사용한다.

## Canonical Root Folders

- `00_class_materials`
- `01_codex_prompts`
- `02_report_templates`
- `03_weekly_reports`
- `04_final_paper`
- `05_references`
- `06_submission`
- `scripts`
- `assets`

## Final Paper Workspace

JKAIS 기말논문 관련 작업 원본은 반드시 `04_final_paper/` 내부에 저장한다.

- 학회 양식: `04_final_paper/00_journal_format/`
- 논문 기획: `04_final_paper/01_planning/`
- 주차별 보고서 반영: `04_final_paper/02_weekly_reflection/`
- 관련연구: `04_final_paper/03_related_work/`
- 데이터/실험/결과: `04_final_paper/04_methodology_experiment/`
- 논문 초안: `04_final_paper/05_draft/`
- 부록/윤리/검증표: `04_final_paper/06_appendices/`
- 제출 전 점검: `04_final_paper/07_final_submission/`

## Final Submission Copy

실제 제출용 복사본은 다음 경로에 둔다.

- `06_submission/2026-06-13_JKAIS_mock_submission/`

## Do Not Create These Root Folders

다음 루트 폴더는 새로 만들지 않는다.

- `paper`
- `data`
- `experiments`
- `outputs`
- `references`
- `submission`
- `ethics`
- `reports`

필요한 경우 반드시 `04_final_paper/` 내부 번호형 하위 폴더 또는 `06_submission/` 아래 제출 폴더를 사용한다.
