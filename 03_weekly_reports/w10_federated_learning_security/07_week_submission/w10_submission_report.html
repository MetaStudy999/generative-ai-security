1. 한 문장 요약

FL 보안 평가는 raw data 비공유 여부만으로 판단할 수 없으며, local update와 aggregation 단계의 privacy·integrity 위험을 global accuracy, global F1, ASR, Privacy Leakage Proxy, aggregation type, communication bytes, reproducibility evidence로 함께 기록해야 한다.

2. 학습 배경과 주차 목표

본 실습은 실제 FL framework, 실제 클라이언트 데이터, 실제 privacy attack을 재현하는 것이 아니라 W10의 핵심인 FL 보안 평가축을 안전하게 설명하기 위한 최소 toy protocol이다. 따라서 synthetic federated binary classification 기반 안전 toy 실험과 toy logistic regression을 사용하였다.

4. 보안 이슈 30% 정리

주요 이슈는 gradient leakage, membership inference, poisoning, model poisoning, backdoor, malicious client, update integrity, privacy policy다. 실제 개인정보, 실제 FL 서비스 침해, 무단 클라이언트 접속, 실제 공격 payload는 제외한다.

11. 기말논문 연결

W10는 분산 AI 학습의 보안 평가 프레임워크를 설명하는 보조 근거로 활용할 수 있다. 단, W10 결과는 toy protocol이므로 실제 FL 보안 성능 근거로 과장하지 않는다.
