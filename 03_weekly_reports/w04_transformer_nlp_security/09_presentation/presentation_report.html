AI 원리와 보안 이슈

Self-attention은 토큰 사이 관계를 계산하고, Q/K/V와 multi-head attention은 문맥 정보를 여러 관점에서 결합한다. Efficient Transformer는 sparse attention, low-rank approximation, kernelized attention 등으로 긴 입력의 계산 비용을 줄인다.

보안 관점에서는 단어 치환 공격, 문장 재구성, prompt privacy, ICL leakage를 clean score, ASR, privacy leakage, utility score로 나누어 평가한다.
