# Contribution 후보

1. LLM/RAG 시스템의 prompt, context, output, code, log, benchmark를 하나의 보안 평가 생명주기로 연결한다.
2. Utility, ASR, privacy leakage, refusal quality, over-refusal, code vulnerability rate를 함께 기록하는 평가표를 제안한다.
3. 실제 공격 재현 없이 synthetic prompt category로 안전하게 평가 프로토콜을 검증하는 절차를 제시한다.
4. Seed, config, CSV, JSON, run log를 보존해 보고서 수치와 실행 근거를 일치시키는 재현성 절차를 제시한다.
