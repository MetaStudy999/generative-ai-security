## 사용 범위 설명

AI는 초안 작성, 표 구조화, 서지 불일치 발견, 실험 산출물 대조, KCI/SCI 전환 섹션 초안 작성에 사용되었다. AI 산출물은 최종 제출본이 아니라 사람이 검토해야 하는 제출용 최종 초안으로 취급한다.

## 안전·윤리 처리

- 실제 LLM/API 호출, 실제 외부 시스템 공격, 실제 의료·금융 시스템 조작, 실제 개인정보 사용, live tool invocation은 수행하지 않았다.
- prompt injection 공격 절차는 악용 가능한 단계별 지침으로 상세화하지 않았다.
- 실험 결과는 synthetic RAG document와 rule-based toy evaluator의 산출물로만 해석한다.
- DOI/URL을 확인하지 못한 문헌 정보는 `확인 필요`로 남겼다.
