# W02 주차 제출 폴더

이 폴더는 대규모 최적화 & 데이터 오염 위협 주차의 제출본과 제출 점검표를 관리한다.

## 제출 기준

- 최종 보고서는 `06_report/final/integrated_report_final.md`를 기준으로 한다.
- 제출용 주차 보고서는 `07_week_submission/w02_submission_report.md`와 `07_week_submission/w02_submission_report.html`을 기준으로 한다.
- 참고문헌 검증은 `01_papers/doi_check.md`를 기준으로 한다.
- 실험 코드는 `04_experiment/src/run_experiment.py`에 있으며, 정량 결과는 Docker 실행 후 `04_experiment/outputs/`에 기록되어 있다.
- 발표자료는 `09_presentation/`의 슬라이드, 발표자 노트, Q&A, 1페이지 요약, HTML을 기준으로 한다.
