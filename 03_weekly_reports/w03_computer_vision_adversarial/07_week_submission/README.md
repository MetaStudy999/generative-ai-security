# W03 주차 제출 폴더

이 폴더는 컴퓨터비전 표현학습 & 비전 대적공격 주차의 제출용 최종 초안과 제출 점검표를 관리한다.

## 구성

| 파일 | 용도 |
|---|---|
| `w03_submission_report.md` | 제출용 보고서 Markdown 원본 |
| `w03_submission_report.html` | 브라우저 열람/인쇄용 제출 보고서 |
| `submit_checklist.md` | 제출 전 점검표 |

## 작성 원칙

- 실험 수치는 `../04_experiment/outputs/run_log.md` 기준으로만 작성한다.
- DOI/URL은 `../01_papers/doi_check.md`의 확인 상태와 맞춘다.
- HTML은 Markdown 제출본과 같은 내용을 담아야 한다.
- 최종 제출 확정은 사람이 PDF 보관 정책과 인용 형식을 검토한 뒤 진행한다.
