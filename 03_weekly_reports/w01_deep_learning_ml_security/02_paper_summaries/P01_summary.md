## 2. 한 문장 요약

> 이 논문은 딥러닝이 다층 표현학습과 역전파를 통해 복잡한 데이터 구조를 학습하는 방식을 정리하며, 보안 연구에서 모델 취약성과 평가 기준을 이해하기 위한 AI 원리의 출발점을 제공한다.

## 3. 연구문제

이 논문은 “왜 깊은 신경망이 이미지, 음성, 자연어 등 고차원 데이터에서 기존 특징공학 중심 방법보다 강력한가”를 설명한다. W01 관점에서는 딥러닝 모델이 어떤 표현을 학습하고, 그 표현이 일반화 성능과 취약성에 어떤 영향을 주는지가 핵심 질문이다.

## 4. 핵심 개념

| 개념 | 설명 | 보안 연구 연결 |
|---|---|---|
| Representation learning | 사람이 설계한 특징 대신 모델이 데이터의 여러 추상 수준을 학습한다. | 공격자는 입력의 작은 변화가 내부 표현을 어떻게 바꾸는지 노린다. |
| Backpropagation | 손실함수의 기울기를 이용해 층별 파라미터를 갱신한다. | gradient 기반 adversarial example의 기술적 기반이 된다. |
| Convolutional network | 지역성, 가중치 공유, 계층적 특징을 이용해 이미지/신호를 처리한다. | 시각 모델의 물리적 교란, 패치 공격, 강건성 평가와 연결된다. |
| Recurrent network | 순차 데이터의 문맥 정보를 모델링한다. | 시퀀스 기반 침입탐지, 로그 분석, 텍스트 공격 분석의 배경이 된다. |

## 5. 방법론

리뷰 논문으로서 딥러닝의 역사, 모델 구조, 학습 원리, 대표 적용 분야를 통합적으로 설명한다. 실험 논문이라기보다 개념과 사례를 연결하는 성격이 강하므로, 본 보고서에서는 수치 결과보다 딥러닝 시스템을 분석하는 기본 용어와 평가 관점을 추출했다.

## 6. 주요 결과

딥러닝은 다층 구조를 통해 원시 입력에서 점진적으로 추상적인 표현을 학습한다. 이 과정은 음성인식, 영상인식, 객체탐지, 자연어처리 등 다양한 영역에서 성능 개선을 이끌었고, 대규모 데이터와 연산 자원, 역전파 기반 학습이 결합될 때 특히 강력하다.

## 7. 보안 관점 분석

보안 관점에서 중요한 점은 딥러닝의 장점과 취약점이 같은 원리에서 나온다는 것이다. 고차원 표현은 복잡한 패턴을 학습하게 해주지만, 입력 분포가 조금만 달라져도 예측이 불안정해질 수 있다. 또한 내부 표현과 출력 확률이 공격자에게 관측될 경우 모델 역추론, membership inference, adversarial perturbation의 단서가 될 수 있다.

## 8. 한계와 오픈문제

이 논문은 딥러닝 일반론을 다루므로 보안 위협모형을 직접 제시하지 않는다. 따라서 기말 논문에서는 P02-P05의 생명주기 보증, 대적 공격, 프라이버시 공격 분류체계를 결합해 보안 평가 기준을 별도로 구성해야 한다.

## 9. 기말 논문에 반영할 부분

기말 논문의 배경 장에서 딥러닝을 “표현학습 기반의 함수 근사 시스템”으로 정의할 때 P01을 핵심 근거로 사용한다. 연구방법 장에서는 일반화 오차, 손실함수, gradient, 입력 표현을 보안 평가 지표와 연결하는 데 활용한다.
