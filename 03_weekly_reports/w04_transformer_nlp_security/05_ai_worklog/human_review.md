# 사람 검토 기록

## 검토 필요 항목

- DOI/URL 공식 출처 대조
- 로컬 PDF와 프롬프트 논문명 일치 여부 확인
- 원문 세부 실험 설정과 수치 검증
- 기말 논문에 실제 인용할 문헌 선별
- AI가 작성한 표현의 과장 여부 점검
- synthetic toy 실험 결과를 실제 모델 성능으로 일반화하지 않는지 확인

## 현재 판단

W04 산출물은 제출용 최종본 형태로 정리되었다. P02/P03은 출판 DOI를 확인했고 P01/P04/P05는 arXiv DOI/URL을 확인했다. 실험값은 `04_experiment/outputs/`의 CSV/JSON/run_log 기준으로 반영했다.
