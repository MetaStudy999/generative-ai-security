## 관리 메모

- 현재 기록은 로컬 PDF 첫 페이지와 확인 가능한 DOI/arXiv 식별자 중심이다.
- 제출 시 원문 PDF 재배포가 아니라 서지정보와 분석 내용만 인용한다.
- `01_papers/pdf/`의 PDF 5개는 `git ls-files` 기준 추적 중이다. public GitHub 저장소에는 원칙적으로 PDF 원문 대신 DOI/URL, 서지정보, 요약만 남기는 것이 안전하다.
- `.gitignore`에는 `03_weekly_reports/**/01_papers/pdf/*.pdf` 규칙이 이미 있으나, 기존 추적 파일에는 자동 적용되지 않는다. 삭제 여부는 사용자 승인 후 결정한다.
