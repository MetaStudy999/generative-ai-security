# W04 발표자료

## 구성

| 파일 | 용도 |
|---|---|
| `presentation_report.md` | 8-10분 발표용 보고서 |
| `presentation_report.html` | 브라우저 확인용 발표 보고서 |
| `presentation_slides.md` | 슬라이드 원본 |
| `presentation_slides.html` | 키보드로 넘길 수 있는 브라우저 슬라이드 |
| `speaker_notes.md` | 슬라이드별 발표자 대본 |
| `qna.md` | 예상 질문과 답변 |
| `one_page_handout.md` | 청중 배포용 1페이지 요약 |

## 기준

정량값은 `../04_experiment/outputs/run_log.md`, `metrics_summary.csv`, `results.json` 기준이다. W04 실험은 synthetic toy 실험이며 실제 Transformer 또는 LLM 성능 주장으로 일반화하지 않는다.
