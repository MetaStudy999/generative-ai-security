# W01 통합보고서 초안

이 초안은 `06_report/final/integrated_report_final.md`의 작성 전개를 보존하기 위한 draft이다. 최종 제출용 내용은 final 문서를 기준으로 한다.

## 핵심 논지

딥러닝/ML 보안 평가는 모델 accuracy만으로 충분하지 않다. W01에서는 딥러닝 원리, ML 생명주기 보증, 침입탐지 지표, 대적 ML, 프라이버시 공격을 연결해 clean performance, robust performance, privacy leakage, reproducibility evidence를 분리해 기록하는 평가 관점을 세운다.

## 주요 구성

1. P01로 딥러닝 표현학습과 일반화 원리를 정리한다.
2. P02로 ML 생명주기와 보증 증거를 정리한다.
3. P03으로 보안 탐지 평가 지표와 공개 데이터셋 한계를 정리한다.
4. P04로 adversarial robustness 평가 기준을 정리한다.
5. P05로 privacy leakage와 threat model을 정리한다.
