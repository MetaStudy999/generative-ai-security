# Contribution 후보

1. 신경망 검증, XAI 설명 안정성, 대적 강건성, 공정성, 재현성을 하나의 AI 보안 평가표로 연결하는 프레임워크를 제안한다.
2. Certified robustness와 empirical robustness를 구분하고, clean accuracy, robust accuracy, explanation stability, certified rate, fairness gap, verification cost를 함께 보고하는 절차를 제시한다.
3. Docker/uv, seed, config, 실행 스크립트, CSV/JSON/run log를 묶어 실험 수치와 보고서 수치를 대조 가능한 형태로 보존하는 재현성 절차를 제시한다.
4. `RELATED` 문헌과 DOI 확인 필요 항목을 숨기지 않고 참고문헌 검증표에 남기는 연구윤리 중심 보고 방식을 제시한다.
