# 00_class_materials

이 폴더는 AI 보안 세미나의 기준 문서를 보관하는 공간이다.

## 보관 대상

- 강의계획서
- 기말고사 보고서 과제 안내문
- 수업 운영 및 평가 기준
- 주차별 논문 패킷 목록
- 제출 기준 및 점검표
- 수식 작성 및 검증 권장 도구

## 사용 원칙

주차별 보고서와 기말논문을 작성할 때 이 폴더의 기준 문서를 먼저 확인한다.

특히 기말고사 보고서는 `final_report_assignment_ai_security.md`와 `final_assignment_summary.md`를 우선 기준으로 삼는다. 강의계획서의 Research Track 항목(연구문제, 위협모형, 평가방법, 재현성, 한계/오픈문제)이 기말논문의 서론, 관련연구, 연구방법, 분석/실험, 보안적 함의로 이어지도록 작성한다.

각 분야별 논문에서 수식 또는 알고리즘이 핵심 기여나 평가방법을 이해하는 데 필요하면, 원문 또는 검증 가능한 표준 수식을 기준으로 기호 정의, 직관적 설명, 보안 해석, 평가 지표 연결을 함께 정리한다. 확인하지 못한 수식은 `확인 필요`로 표시하고 임의로 작성하지 않는다.

수식 작성 형식과 권장 도구는 `math_formula_toolchain.md`를 기준으로 한다. 기본 작성은 Markdown + LaTeX math, HTML 렌더링은 KaTeX/MathJax, 문서 변환은 Pandoc, 수식 검산은 필요 시 `sympy`를 사용한다.
