# 보안 이슈 30% 정리

## 1. 주요 보안 이슈

W04의 보안 이슈는 NLP 대적공격, 프롬프트 프라이버시, ICL 위험이다. 이 부분은 전체 보고서의 30% 비중으로 두되, AI 원리를 실제 위협모형과 평가방법으로 전환하는 데 집중한다.

- NLP 대적공격
- 단어 치환 공격
- 문장 재구성 공격
- 의미 보존 공격
- Prompt privacy
- In-context learning 환경의 민감정보 노출
- Privacy-preserving prompt engineering
- LLM 입력 프롬프트의 보안 통제
