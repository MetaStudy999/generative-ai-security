# W08 기말논문 주제 후보

| 우선순위 | 주제 후보 | 장점 | 주의점 |
|---:|---|---|---|
| 1 | RAG 기반 생성형 AI 시스템에서 간접 프롬프트 인젝션 대응 방안 연구 | W08 문헌과 실험이 직접 연결됨 | 실제 공격 payload 없이 안전한 synthetic 평가로 제한 |
| 2 | 보안형 RAG 시스템에서 문서·graph 출처 검증 메타데이터 설계 연구 | GraphRAG 문헌과 source verification 지표 연결 | graph provenance 설계 범위를 명확히 해야 함 |
| 3 | LLM 에이전트의 tool 권한 오남용 방지를 위한 human approval gate 연구 | agentic RAG와 운영 통제 연결 | 사용성 저하와 보안성 균형 분석 필요 |
| 4 | Safety-critical RAG 응용에서 prompt injection 평가 체크리스트 연구 | 의료 LLM 취약성 논문과 연결 | 의료 세부 도메인 지식은 과도하게 확장하지 않음 |

## 추천

1순위 주제를 중심으로 하고, 2순위의 출처 검증과 3순위의 approval gate를 방법론 하위 요소로 포함하는 방식이 가장 안정적이다.
