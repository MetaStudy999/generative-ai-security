## PDF 저작권 위험 메모

- `git ls-files 03_weekly_reports/w06_diffusion_gan_deepfake/01_papers/pdf` 기준 PDF 5개가 이미 git 추적 대상이다.
- `.gitignore`에는 `03_weekly_reports/**/01_papers/pdf/*.pdf` 규칙이 이미 존재하지만, 이미 추적 중인 파일에는 적용되지 않는다.
- public GitHub 저장소에 출판사 PDF 원문이 올라가면 저작권 위험이 있다. 삭제는 사용자 승인 없이 수행하지 않으며, 최종 제출 전 사람이 삭제 또는 비공개 저장소 전환 여부를 결정해야 한다.
