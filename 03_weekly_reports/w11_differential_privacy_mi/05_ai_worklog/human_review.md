# 사람 검토 기록

## 검토 완료 항목

- W11 공통 산출물 구조 확인
- 로컬 PDF와 지정 논문 불일치 표시
- 실험 결과와 `outputs/` 수치 일치 확인
- `epsilon_proxy`를 실제 DP 보장으로 표현하지 않도록 수정
- 제출용/발표용 산출물 생성 확인

## 최종 제출 전 추가 검토 필요

- P01/P02/P04의 ACM DOI 공식 출처 대조
- P03/P05 지정 논문 원문 PDF 확보
- 기말 논문에 실제 인용할 문헌 선별
- 정식 DP-SGD/accountant 실험으로 확장할지 여부 판단

## 현재 판단

W11 산출물은 제출 가능한 구조와 실행 로그를 갖춘 상태다. 다만 참고문헌 공식 DOI와 대체 PDF 문제는 최종 제출 또는 기말논문 작성 전 추가 확인이 필요하다.
