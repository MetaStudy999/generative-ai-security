# 3. 큰 흐름으로 보면

이 강의계획서는 다음 순서로 난이도가 올라갑니다.

1. **Week 1~3:** 딥러닝·최적화·컴퓨터비전 기초와 ML 보안 분류
2. **Week 4~6:** Transformer, 자기지도학습, 생성모형, 딥페이크
3. **Week 7~9:** LLM, RAG, 프롬프트 인젝션, 심층강화학습
4. **Week 10~12:** 연합학습, 차등프라이버시, 신경망 검증, XAI
5. **Week 13~15:** 모델 IP, MLOps 보안, 재현성, 논문 작성·평가 전략

즉, 단순한 AI 보안 개론이 아니라 **박사과정 논문지도형 세미나**에 가깝고, 각 주차의 논문을 “요약”하는 수준을 넘어 **연구문제, 위협모형, 평가방법, 재현성, 한계·오픈문제**로 구조화하도록 설계되어 있습니다.
