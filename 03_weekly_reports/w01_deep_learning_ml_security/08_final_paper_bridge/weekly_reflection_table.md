# 주차별 성찰표

| 항목 | 내용 |
|---|---|
| 주차 | W01 |
| 주제 | 딥러닝 패러다임 & ML 보안 분류학 |
| 핵심 학습 | 딥러닝은 표현학습과 일반화의 문제이며, 보안 평가는 모델 단독이 아니라 생명주기 전체의 문제다. |
| AI 원리 | 다층 표현학습, 역전파, 일반화, overfitting, 모델 평가 |
| 보안 연결 | ML 생명주기 보증, 침입탐지 지표, adversarial robustness, privacy leakage |
| 가장 중요한 깨달음 | Clean accuracy가 높아도 robust accuracy나 privacy safety가 낮으면 보안성 주장을 할 수 없다. |
| 기말논문 연결 | 생명주기 기반 AI 보안 평가 프레임워크의 기준 주차로 사용한다. |
| 보완 필요 | synthetic toy 결과를 실제 시스템 평가와 구분해 해석한다. |
| 검증 상태 | DOI/URL은 확인 가능한 범위에서 갱신했고, 실험값은 실행 로그 기반으로 기록했다. |
