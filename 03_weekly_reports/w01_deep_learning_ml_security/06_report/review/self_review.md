## 남은 리스크

- P04는 arXiv 기준으로 정리했으므로 최종 출판 버전이 있다면 참고문헌 형식을 갱신해야 한다.
- 정량 실험값은 synthetic toy data 기준이므로 실제 시스템 보안성으로 일반화하지 않아야 한다.
- 원문 페이지 단위 인용이 필요하면 PDF를 직접 열어 페이지와 문장을 확인해야 한다.
