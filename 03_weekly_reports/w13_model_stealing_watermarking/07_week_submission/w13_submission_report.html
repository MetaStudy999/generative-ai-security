초록

본 보고서는 모델 지식재산, 모델 도난, 모델 추출, 워터마킹, 핑거프린팅을 AI 원리와 보안 이슈로 나누어 정리한다. synthetic toy classifier 실험에서 query budget 100/500/1000의 extraction fidelity는 0.864000/0.920000/0.902000, watermark detection은 0.700000/1.000000/1.000000이었다. false positive proxy는 0.600000으로 높아, 본 결과는 소유권 검증의 필요성과 한계를 함께 보여주는 교육용 결과다.

키워드: model stealing, model extraction, watermarking, fingerprinting, extraction fidelity, false positive

AI 원리 70%와 보안 이슈 30%

모델 IP는 파라미터, 구조, 학습 데이터, 출력 행동, 생성물 출처를 포함한다. 모델 추출은 query-response 정보로 substitute model을 학습하는 절차이며, 워터마킹과 핑거프린팅은 추출 이후 소유권 검증을 돕는다. 보안 평가는 confidentiality, integrity, availability, privacy, safety, accountability 관점에서 수행한다.
