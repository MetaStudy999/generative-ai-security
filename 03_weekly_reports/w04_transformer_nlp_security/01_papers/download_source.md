## PDF 보관 정책 메모

- `git ls-files` 기준 PDF 5개가 이미 Git 추적 대상으로 확인되었다.
- `.gitignore`에는 `03_weekly_reports/**/01_papers/pdf/*.pdf` 규칙이 이미 존재한다.
- 공개 GitHub 저장소에는 출판사 PDF 원문을 올리지 말고, DOI/URL, 서지정보, 요약, 개인 독해 메모만 남기는 방식을 권장한다.
- 사용자의 명시 승인 없이 PDF를 삭제하지 않았다. 이미 커밋된 PDF는 제출 전 삭제 또는 저장소 비공개 전환 여부를 사람이 결정해야 한다.
