# PDF 원문 보관 정책

## 1. 원칙

공개 GitHub 저장소에는 저작권이 있는 논문 원문 PDF를 직접 보관하지 않는 것을 원칙으로 한다.

## 2. 허용

- DOI
- 공식 출판사 landing page URL
- arXiv 등 공개 배포가 허용된 URL
- 개인 로컬 보관 여부 표시
- 논문 요약 및 본인의 분석

## 3. 주의

- 출판사 유료 PDF 원문을 public repository에 업로드하지 않는다.
- 이미 추적 중인 PDF는 삭제 여부를 사용자 확인 후 처리한다.
- 삭제 전에는 DOI 목록과 공식 URL을 먼저 정리한다.

## 4. 대체 관리 방식

논문 원문 대신 다음 파일에서 관리한다.

- `05_references/doi_index.md`
- `04_final_paper/06_appendices/reference_verification.md`
