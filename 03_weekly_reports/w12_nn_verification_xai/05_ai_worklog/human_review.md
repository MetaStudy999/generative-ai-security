# 사람 검토 기록

## 검토 필요 항목

- DOI/URL 공식 출처 대조
- 로컬 PDF와 프롬프트 논문명 일치 여부 확인
- `RELATED` 문헌의 대체 타당성 확인
- 원문 세부 실험 설정과 수치 검증
- 기말 논문에 실제 인용할 문헌 선별
- AI가 작성한 표현의 과장 여부 점검

## 현재 판단

W12 산출물은 제출 가능한 구조와 실행 산출물을 갖춘 상태다. 실험 수치는 synthetic toy 환경에서 직접 생성된 `outputs/` 값만 반영했다. 다만 참고문헌 DOI/URL과 원문 세부 내용은 최종 제출 전 사람이 공식 출처와 대조해야 한다.
