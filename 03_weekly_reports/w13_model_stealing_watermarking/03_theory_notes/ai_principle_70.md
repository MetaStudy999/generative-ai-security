# AI 원리 70% 정리

## 1. 핵심 이론

W13의 AI 원리는 Model IP, model stealing, model extraction이다. 이 주차에서는 보안 이슈를 먼저 끌어오지 않고, 모델 또는 시스템이 어떤 학습 구조와 평가 구조를 갖는지 이해하는 데 70%의 비중을 둔다.

핵심 항목은 다음과 같다.

- 모델 지식재산권의 개념
- 모델 도난과 모델 추출의 차이
- 공개 API 기반 모델 추출 구조
- Query-response 기반 대체 모델 학습
- Model fingerprinting
- Model watermarking
- DNN watermarking의 기본 구조
- Robust watermark와 adaptive watermark
- GAN/생성모형에서의 공격·방어 구조
- Model utility와 protection trade-off

## 3. 수식 또는 알고리즘

```text
입력/데이터 정의 -> 모델 지식재산권의 개념 -> 모델 도난과 모델 추출의 차이 -> 공개 API 기반 모델 추출 구조 -> Query-response 기반 대체 모델 학습 -> 모델 또는 시스템 출력 -> 평가 지표 산출
```

수식 수준에서는 학습 목적함수, 평가 지표, 일반화 또는 효율성 조건을 분리해 기록한다. 세부 수식은 원문 확인 후 최종 논문에 필요한 것만 엄선한다.

## 4. 초보자용 설명

모델 지식재산(IP)/모델 도난/모델 추출 위협를 공부할 때 먼저 볼 것은 "모델이 무엇을 입력으로 받고, 무엇을 학습하며, 어떤 기준으로 성공을 판단하는가"이다. 이 구조를 이해해야 공격자가 어느 지점을 건드릴 수 있는지도 보인다.

## 5. 보안 연구와의 연결

AI 원리의 취약 지점은 곧 보안 평가 항목이 된다. 모델 지식재산권의 개념, 모델 도난과 모델 추출의 차이, 공개 API 기반 모델 추출 구조를 이해하면 Model stealing, Model extraction, API query abuse가 왜 발생하는지 설명할 수 있다.
