# W03 발표자료

## 구성

| 파일 | 용도 |
|---|---|
| `presentation_report.md` | 발표 흐름, 핵심 주장, 논문 역할, 실험 근거, 예상 질문을 통합한 발표용 보고서 |
| `presentation_report.html` | 브라우저 열람/인쇄용 발표 보고서 |
| `presentation_slides.md` | 발표 슬라이드 원본 |
| `presentation_slides.html` | 키보드로 넘길 수 있는 브라우저 발표용 슬라이드 |
| `speaker_notes.md` | 슬라이드별 발표자 대본 |
| `qna.md` | 예상 질문과 답변 |
| `one_page_handout.md` | 청중 배포용 1페이지 요약 |

## 발표 시간 기준

- 권장 발표 시간: 8-10분
- 권장 슬라이드 수: 10-12장
- 핵심 메시지: 비전 모델의 성능 평가는 clean accuracy와 robust accuracy, ASR, 재현성 근거를 분리해서 기록해야 한다.

## 작성 원칙

- 실험 수치는 `04_experiment/outputs/run_log.md` 기준으로만 사용한다.
- DOI/URL은 확인되었지만 원문 세부 수치, PDF 보관 정책, 실제 모델 일반화는 확정 표현으로 말하지 않는다.
- 실제 개인정보, 운영 서비스 이미지, 무단 공격 절차는 발표 범위에서 제외한다.

## 사용 방법

1. `presentation_slides.html`을 브라우저로 연다.
2. 키보드 방향키, PageUp/PageDown, Space, Home/End로 슬라이드를 이동한다.
3. 발표 전 `speaker_notes.md`와 `qna.md`로 발화 흐름과 예상 질문을 점검한다.
4. 청중 배포가 필요하면 `one_page_handout.md`를 사용한다.
5. 인쇄 또는 PDF 저장이 필요하면 브라우저 인쇄 기능을 사용한다.
