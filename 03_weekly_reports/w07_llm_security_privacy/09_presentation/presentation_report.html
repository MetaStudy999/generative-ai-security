결론

  1. LLM 보안 평가는 다중지표 문제다.
  2. ASR 단독 평가는 utility와 over-refusal을 놓친다.
  3. Utility 단독 평가는 leakage와 code risk를 놓친다.
  4. 실행 로그 없는 수치는 주장하지 않는다.
